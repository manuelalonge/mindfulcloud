Install CLI

Requirements

  • Windows 10, Mac OSX, Linux
  • Node.js v12 or higher

Install

Mac (Homebrew)

$ brew install mindfulcloud-cli

NPM

$ npm install mindfulcloud-cli

Yarn

$ yarn install mindfulcloud-cli
